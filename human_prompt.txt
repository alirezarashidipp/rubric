Thank you for participating in this annotation exercise. Your assessments will be used to evaluate and improve the quality of an automated Jira story assessment model. Please review each Jira ticket carefully, identify the user story description and acceptance criteria from the ticket body, and assign scores based only on explicit evidence in the ticket. Do not guess missing information. If something is unclear, missing, or only implied, choose the lower score. Use 0 when there is no assessable evidence for a 0–4 metric. For each score, provide a short rationale explaining the main evidence or missing information. Consistent and accurate scoring is important for improving model reliability.
